## 問題1: 製品と価格

以下の3つのタイプの製品(product)を売る会社を考える

- ワードプロセッサ
- スプレッドシート
- データベース

製品は名前(name)、タイプ(type)、価格(price)を持つ

例:

- MS Word の価格は18,800円、タイプは `'W'`
- MS Excel の価格は 27,800円、タイプは `'S'`
- MS SQL Server の価格は 919,000円、タイプは `'D'`
